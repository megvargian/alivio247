🔧 ALIVIO247 Email Setup Guide

Since PHP mail() isn't working on your server, here's how to set up SMTP email:

📧 Step 1: Test Current Setup

Visit your email-config.php file:

yourdomain.com/email-config.php

This will show if PHP mail() is working or not.

🔑 Step 2: Set Up Gmail App Password

  1. Go to your Gmail account (alivio247alivio@gmail.com)
  2. Click Account Settings → Security
  3. Enable "2-Step Verification" if not already enabled
  4. Go to "App passwords"
  5. Generate an app password for "Mail"
  6. Copy the 16-character password (e.g., abcd efgh ijkl mnop)

⚙️ Step 3: Update SMTP Configuration

Edit the file contact-handler-smtp.php and update this line:

'password' => 'YOUR_GMAIL_APP_PASSWORD', // Replace with the app password from Step 2

Replace YOUR_GMAIL_APP_PASSWORD with the actual app password you generated.

⚠️ Alternative Solution: Use cPanel Email

If Gmail SMTP doesn't work, you can use your hosting provider's email:

  1. Go to cPanel → Email Accounts
  2. Create or note the password for support@alivio247.com
  3. Update the SMTP settings to use your domain's mail server

📮 Step 4: Test the Contact Form

  1. Visit your website's contact page
  2. Fill out the form with your own email
  3. Submit and check if you receive the emails
  4. Check spam folders too

✅ Expected Results:

  • Customer receives a "Thank You" email
  • support@alivio247.com receives notification
  • alivio247alivio@gmail.com receives notification

🚨 If Still Not Working:

The system will create a contact-results.html file showing the submitted message, so you won't lose any customer inquiries.

Contact your hosting provider and ask them to:

  • Enable PHP mail() function
  • Check email server configuration
  • Verify domain DNS/MX records
