📧 ALIVIO247 GoDaddy Email Setup

Complete setup guide for GoDaddy hosted email

✅ Step 1: Create/Verify Email Account

Since you already have info@alivio247.com set up, you can either:

Option A: Use existing info@alivio247.com

Keep using your current email account with the password you already have.

Option B: Create support@alivio247.com (Recommended)

  1. Log into your GoDaddy account
  2. Go to "My Products"
  3. Find your domain and click "Email"
  4. Create a new email: support@alivio247.com
  5. Set a strong password and note it down

⚙️ Step 2: GoDaddy SMTP Settings (Already Updated)

I've already updated your contact-handler-smtp.php with GoDaddy's SMTP settings:

$smtpConfig = [ 'host' => 'smtpout.secureserver.net', // GoDaddy's SMTP server 'port' => 587, // GoDaddy's TLS port 'encryption' => 'tls', // TLS encryption 'username' => 'support@alivio247.com', // Your email account 'password' => 'YOUR_SUPPORT_EMAIL_PASSWORD', // Update this! 'from_email' => 'support@alivio247.com', 'from_name' => 'ALIVIO247 Contact Form' ];

🔑 Step 3: Update the Password

You need to update ONE LINE in the contact-handler-smtp.php file:

'password' => 'YOUR_ACTUAL_EMAIL_PASSWORD', // Replace with real password

If using support@alivio247.com: Use the password you set when creating it

If using info@alivio247.com: Use your existing password and change the username/from_email to info@alivio247.com

🔄 Alternative: Keep Using info@alivio247.com

If you prefer to use your existing email, change these lines:

'username' => 'info@alivio247.com', // Instead of support@ 'from_email' => 'info@alivio247.com', // Instead of support@ 'password' => 'j*z}(n~.}mU4bIKR', // Your existing password

✅ GoDaddy Email Advantages:

  • Reliable Delivery: GoDaddy has excellent email delivery rates
  • Professional: Emails come from your domain (not Gmail)
  • Integrated: Works seamlessly with your hosting
  • No Extra Setup: No need for app passwords or 2FA

🧪 Step 4: Test Your Setup

  1. Update the password in contact-handler-smtp.php
  2. Upload the file to your website
  3. Visit your contact form
  4. Submit a test with your own email
  5. Check for emails in both inboxes

🔍 GoDaddy Email Settings Reference:

  • SMTP Server: smtpout.secureserver.net
  • Port: 587 (TLS) or 465 (SSL)
  • Security: TLS or SSL/TLS
  • Authentication: Required
  • Username: Your full email address

📞 Expected Email Flow:

  1. Customer submits form → Gets "Thank You" email
  2. support@alivio247.com → Gets notification with customer details
  3. alivio247alivio@gmail.com → Gets same notification
  4. All emails sent from: support@alivio247.com (professional!)

✅ Why GoDaddy Email is Better:

Using support@alivio247.com instead of Gmail makes your business look more professional and trustworthy. Customers see emails coming from your official domain, not a generic Gmail account.
